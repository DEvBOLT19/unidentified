🌍 BOLT EARTH LIVE

A real-time 3D global airspace intelligence dashboard powered by live public aviation data.

🚀 Overview

BOLT EARTH LIVE is a web-based 3D Earth visualization system that plots live aircraft positions across the globe using publicly available aviation data.

The project delivers a cinematic, command-center experience while remaining fully legal and data-transparent.

It runs entirely in the browser and requires no backend.

✨ Core Features

🌐 Data Source

Aircraft data is provided by the OpenSky Network public API.

This project does not:

Access private CCTV systems

Access passenger lists

Access restricted aviation systems

Bypass any protected infrastructure

It uses only publicly broadcast aircraft transponder data.

The concept is similar in principle to platforms such as:

Flightradar24

FlightAware
